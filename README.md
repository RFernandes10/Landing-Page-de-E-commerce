- # React + Vite

This project is a modern and responsive e-commerce landing page built with **React** and **Tailwind CSS**, using **Vite** as the build tool. It provides a minimal setup for React with fast refresh and optimized performance.

## 📌 Features
- 🚀 **Fast and Lightweight** – Optimized for performance with Vite.
- 🎨 **Responsive UI** – Mobile-friendly design with Tailwind CSS.
- 🌙 **Dark Mode Support** – Toggle between light and dark themes.
- 🔧 **Modular Components** – Organized and reusable React components.
- 🏷️ **Dynamic Banners** – Customizable promotional banners via JSON.

## 🛠 Tech Stack
- [React.js](https://reactjs.org/) – A JavaScript library for building user interfaces.
- [Vite.js](https://vitejs.dev/) – A fast build tool and development server.
- [Tailwind CSS](https://tailwindcss.com/) – A utility-first CSS framework for rapid styling.
- [JavaScript (ES6+)](https://developer.mozilla.org/en-US/docs/Web/JavaScript) – Modern JavaScript features.
